解决国内打不开github

发表于 2022-05-07 分类于 git

2，复制任意一个ip
3，打开电脑文件
  • C:\Windows\System32\drivers\etc下的host文件
4，在host文件的最后一行加入刚复制的ip
  • 20.205.243.166 github.com
5,刷新dns
  • cmd窗口执行命令 ipconfig/flushdns
