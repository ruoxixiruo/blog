msyql中的json字段

  • 创建json

  • 插入json

    • 直接插入字符串

    • 使用json函数

  • 查询json

    • 使用json作为条件查询

    • 查询JSON中的数据用 column->path的形式，其中对象类型path的表示方式 .path，数组类型的表示方式[index];

    • 注意（MySQL：5.7.17）：JSON中的元素搜索是严格区分变量类型的，比如整型和字符串时严格区分的（->的形式），但是：->>类型是不区分的

    • 可以使用JSON_CONTAINS函数，但是该函数的第二个参数不接受整型，无论JSON元素时整型还是字符串

  • 更新json

    • 更新整个JSON，与插入时类似

    • JSON_INSERT 插入新值，但如果插入的key在json中已经存在，不会覆盖

    • JSON_SET 插入新值，但如果插入的key在json中已经存在，会覆盖

    • JSON_REPLACE

    • JSON_REMOVE

    • JSON_ARRAY_APPEND 数组中增加元素

    • JSON_ARRAY_INSERT 数组中添加元素

  • 创建索引

    • MySQL的JSON格式数据不能直接创建索引，但是可以变通一下，把要搜索的数据单独拎出来，单独一个数据列，然后在这个字段上创建一个索引
