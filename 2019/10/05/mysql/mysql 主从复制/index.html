发表于 2019-10-05 分类于 mysql

一，主从复制如何工作

  1. master将改变记录到二进制日志(binary log)中
  2. slave将master的binary log events拷贝到它的中继日志(relay log)；
  3. slave重做中继日志中的事件，将改变反映成它自己的数据
